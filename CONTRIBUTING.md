# Contributing

このプロジェクトへの貢献を歓迎します！

## 貢献の方法

### 🐛 バグ報告

Issue を開いてバグを報告してください。
再現手順と、期待される動作・実際の動作を記載してください。

### 💡 機能リクエスト

新しい部署の追加、キャラクター設定の改善、サンプルデータの充実など、
アイデアがあれば Issue で提案してください。

### 🔧 プルリクエスト

1. リポジトリを Fork する
2. フィーチャーブランチを作成する（`git checkout -b feat/add-tax-advisor`）
3. 変更をコミットする（`git commit -m 'feat: 税理士エージェント追加'`）
4. ブランチを Push する（`git push origin feat/add-tax-advisor`）
5. プルリクエストを開く

## ガイドライン

- ファイル名は kebab-case で統一（`sample-apartment.md`、`sample-shinkin.md`）
- 部署 CLAUDE.md は「役割／口調／ルール／フォルダ構成／判断軸」の4セクション構成を維持してください
- 個人情報・実在の人物名・特定企業名を含まないようにしてください
- サンプルデータは架空のものに限定してください

## カスタマイズ事例の共有

「こんな部署を追加しました」「キャラ名をこう変えたら使いやすかった」など、
カスタマイズ事例を Discussions で共有してくれると嬉しいです。
